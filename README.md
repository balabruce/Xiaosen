# Xiaosen

## kline.html — K線畫線練習

一個純前端、單檔的 K 線價量圖模擬器，會隨機出題讓你在圖上練習畫各種技術線。

**開啟方式**：用瀏覽器直接打開 `kline.html`（手機、電腦皆可，支援觸控拖曳）。

**題型**
- 水平壓力線 / 水平支撐線
- 上升趨勢線（支撐）/ 下降趨勢線（壓力）
- 上升通道 / 下降通道（兩條平行線）
- 頸線：雙頂 M 頭 / 雙底 W 底

**玩法**
1. 看題目要你畫哪種線。
2. 在圖上用手指／滑鼠拖曳畫線（可畫多條，通道要畫兩條）。
3. 按「對答案」給你 0–100 分並提示偏高／偏低。
4. 「看答案」用藍色虛線疊上正解對照學習。

其他功能：MA5/MA20 均線切換、水平吸附、復原／清除、練習題數與均分統計。題目每次隨機生成，價量與觸價點都是刻意設計的，方便對照學習。
